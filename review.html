Block elements

Block elements are a significant part of the page and may contain other block and inline elements. This text is inside of the block element, p.

Headings are also block elements.

Html holds the content of the page and tags describe meaning of the content

below is an hr tag, creating a horizontal line, in action

Inline elements

Inline elements are smaller items, and must be inside of the block element. Inline elements cannot contain block elements, and can contain other inline elements.

Inline elements are

  1. img - images
  2. a - links
  3. br - line breaks
  4. em - emphasis, often italics
  5. strong - stron emphasis, often bold

Lists
